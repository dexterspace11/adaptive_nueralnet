# Adaptive NeuralNet v0.2 🚀

A lightweight, efficiency-aware neural network for fast and resource-conscious forecasting — perfect for time series like BTC prices.

## Features

- 🔄 Adaptive parameter compression (based on importance)
- ⚙️ Efficiency-aware logic to adapt precision or complexity
- 🧠 Simple feedforward neural network, fast to train
- 📉 Built-in training loop with live loss monitoring
- 📦 Works well for financial forecasting tasks
